# 对话点数指示器

随着聊天记录的增加，你会发现页面底部有一小行字和一个小圆圈也在发生变化，这个小组件告诉了你当前对话的长度是否还在模型的承载能力之内。

据开发人员观察，当页内点数用量达到100%时，模型并不会立即终止输出，一般会有10% - 30%的冗余，冗余量用完之后，模型才会拒绝输出，不过你随时可以在侧栏的高级功能中选择承载能力更强的对话模型。

以下是三个OpenAI文本模型的点数承载能力：

* GPT-3.5-turbo: `16000`
* GPT-4: `8000`
* GPT-4-turbo: `128000`
